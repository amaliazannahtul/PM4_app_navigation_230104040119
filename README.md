## Deskripsi
Praktikum ini bertujuan untuk memahami konsep navigasi aplikasi
Android menggunakan Jetpack Compose, meliputi:
- Navigasi antar screen
- Explicit dan implicit navigation
- Pengiriman data antar screen
- Back stack
- Konsep Hub / Fragment style navigation
